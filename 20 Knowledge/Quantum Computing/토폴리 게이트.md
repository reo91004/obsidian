# 토폴리 게이트, Toffoli Gate
> 이걸 ==CCNOT 게이트==라고 부르기도 한다.

토폴리 게이트는 3개의 입력$(A,B,C)$과 3개의 출력$(A',B',C')$을 가진다. 동작 원리는 매우 직관적이다.

- **제어 비트 (A, B):** 첫 번째와 두 번째 비트는 제어 역할을 한다. 이 값들은 출력에서 변하지 않고 그대로 유지된다 $(A′=A,B′=B)$.
- **대상 비트 (C):** 세 번째 비트는 대상(Target)이다. **앞의 두 제어 비트가 모두 1일 때만** 세 번째 비트의 값을 뒤집는다(NOT 연산). 그 외의 경우에는 원래 값을 유지한다.

| **입력 (A, B, C)** | **출력 (A', B', C')** | **비고**            |
| ---------------- | ------------------- | ----------------- |
| 0, 0, 0          | 0, 0, 0             | 변화 없음             |
| 0, 1, 0          | 0, 1, 0             | 변화 없음             |
| 1, 0, 0          | 1, 0, 0             | 변화 없음             |
| **1, 1, 0**      | **1, 1, 1**         | **C 반전 (0 -> 1)** |
| 1, 1, 1          | 1, 1, 0             | **C 반전 (1 -> 0)** |

#### **가역성 (Reversibility)**

일반적인 NAND나 AND 게이트는 결과값만 보고 입력값이 무엇이었는지 완벽하게 알아낼 수 없다(정보의 손실 발생). 하지만 토폴리 게이트는 출력을 다시 입력으로 넣으면 원래의 값이 나온다. 즉, **정보 손실이 없는 가역 계산**이 가능하다.

#### **범용성 (Universality)**

앞서 언급한 NAND 게이트처럼, 토폴리 게이트 하나만 있으면 모든 종류의 논리 연산을 구현할 수 있다.

- C의 초기값을 1로 고정하면, 출력 $C'$은 $1 \oplus (A \cdot B)$가 되어 **NAND 게이트**처럼 동작한다.
- A를 1로 고정하고 B를 제어 비트로 쓰면 **CNOT(XOR) 게이트**가 된다.

> **==양자 역학의 물리적 법칙은 가역적이어야 하기 때문에 중요하다.==**
