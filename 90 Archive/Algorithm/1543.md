# 실패했던 것

처음에 완전 비효율적으로 했다가 갈아치웠다.

```python
if __name__ == "__main__":
    import sys

    N = sys.stdin.readline().strip()
    target = sys.stdin.readline().strip()
    cnt = 0
    len_N = len(N)
    len_target = len(target)

    while len_N >= len_target:
        if target in N:
            index = N.find(target)
            cnt += 1
            N = N[index + len_target :]
            len_N = len(N)

    print(cnt)
```

아래가 성공한 코드

```python
if __name__ == "__main__":
    import sys

    document = sys.stdin.readline().strip()
    word = sys.stdin.readline().strip()
    count = 0

    i = 0
    while i <= len(document) - len(word):
        if document[i : i + len(word)] == word:
            count += 1
            i += len(word)  # 찾은 단어를 건너뛰고 계속 검색
        else:
            i += 1

    print(count)
```
